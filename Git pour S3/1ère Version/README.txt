Voici le fichier qui va vous permettre de lancer notre application de Quiz pour tester vos connaissances.

Les étapes :
	1. Vous devez télécharger tout le dossier zippé puis le dézipper avec WinRar ou 7zip par exemple.

	2. Après, vous devez aller dans le dossier Uwamp_2020 puis lancer Uwamp.exe (cette opération lancera un 
serveur sur votre ordinateur, ce qui permettra de gérer les bases de données).

	3. Ensuite, vous devez faire un clic-droit sur le fichier "Lancement.cmd" puis le modifier en remplaçant 
"C:\Adresse\a\remplir" par le chemin vers "javafx-sdk-11.0.2\lib" qui est disponible dans le dossier zippé.

! Attention ! N'oubliez pas de sauvegarder après avoir effectué la modification !

	4. Enfin, vous pouvez lancer le programme "Lancement.cmd" en double-cliquant dessus.

Notes :
	Ceci n'est pas la version finale, il reste des bugs et certains boutons ne font rien, c'est en cours de
développement.
